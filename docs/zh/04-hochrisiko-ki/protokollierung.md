# 记录保存——第12条

## 要求

高风险AI系统在技术上必须设计为能够**自动记录事件**（日志记录）。

## 最低要求

自动日志记录必须涵盖：

- **使用时段**（每次使用的开始/结束）
- **参考数据库**——用于校验输入数据
- 导致搜索的**输入数据**
- 参与人类监督的**人员身份标识**

## 保存期限

- 提供者：**在其控制范围内**保留日志
- 部署者：日志至少保留**6个月**（第26条第6款）

## BAUER GROUP 实施方案

| 措施 | 实施方式 |
|---|---|
| 自动日志 | 在现有日志基础设施中进行结构化日志记录（JSON） |
| 保存 | 保留策略 ≥ 6个月（已通过CRA要求覆盖） |
| 访问 | 日志可应主管机关要求提供 |

::: tip 最低工作量
如果系统已具备结构化日志记录（这是BAUER产品的标准配置），额外工作量仅限于确保包含AI Act特定字段。
:::
