# 标注义务

## 所有标注义务概述

| 内容 | 责任方 | 方式 | 时间 |
|---|---|---|---|
| AI交互 | 提供者 + 部署者 | 可见提示 | 交互开始前/开始时 |
| 合成内容 | 提供者 | 机器可读（C2PA、IPTC等） | 生成时 |
| 深度伪造 | 部署者 | 清晰且可见 | 发布时 |
| 情绪识别 | 部署者 | 告知相关人员 | 部署前 |
| 生物特征分类 | 部署者 | 告知相关人员 | 部署前 |

## 无需标注的情形

- **最低风险**的AI系统（垃圾邮件过滤器、推荐算法等）
- 不对外发布内容的内部AI工具
- AI辅助代码（不属于监管意义上的"合成内容"）
