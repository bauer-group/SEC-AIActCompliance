# 奥地利

奥地利在 **RTR**（Rundfunk und Telekom Regulierungs-GmbH）设立了一个 **AI服务处**（KI-Servicestelle），作为国家级AI能力与信息中心（法律依据包括 KommAustria-Gesetz §20c、TKG 2021 §194a）。

- AI服务处支持机构、企业和公民社会落实 AI Act。
- 政府计划将其**扩展为具有更广泛职权的完整市场监管机构**。
- 须在2026年8月2日前至少设立一个 **AI监管沙盒**（第57条）。

::: warning 状态
最终的机构架构（完整市场监管）仍在建设中（截至2026年6月）。
:::

::: tip BAUER GROUP 相关性
在奥地利市场投放AI系统时，RTR 的 AI服务处是中央联系点；完整的市场监管机构正在建立中。
:::
