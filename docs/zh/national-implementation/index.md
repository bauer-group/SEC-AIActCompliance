# 国家层面落实

## 法规 vs. 国家层面落实

AI Act 是一部**法规**（EU) 2024/1689，因此在所有成员国**直接适用**——无需转化为国内法。但各成员国需在国内层面规定：

- **主管机构的指定**（市场监管 + 公告，第70条），
- **罚款程序**及补充性制裁规则（第99条），
- 至2026年8月2日前至少设立一个**AI监管沙盒**（第57/58条）。

## 欧盟范围内的进展（截至2026年6月）

机构指定的截止日期为**2025年8月2日**。然而截至2026年中期，仅约**27个成员国中的8个**正式指定了主管机构；大多数程序仍在进行中。

| 国家 | 模式 | 中央机构 | 状态 |
|---|---|---|---|
| 德国 | 集中式 | 联邦网络管理局（BNetzA） | KI-MIG 在议会审议（联邦议院 2026年6月11日通过） |
| 西班牙 | 专设机构 | AESIA | 自2024年6月起运作 |
| 爱尔兰 | 分散式 | 国家AI办公室 + 15个机构 | 自2025年9月起运作 |
| 法国 | 分散式 | CNIL / DGCCRF / ARCOM（草案） | 尚未正式指定 |
| 奥地利 | 能力中心 | RTR / AI服务处 | 计划扩展为完整机构 |
