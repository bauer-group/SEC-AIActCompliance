# 被禁止的AI实践 – 第5条

## 概述

自**2025年2月2日**起，欧盟禁止了特定的AI实践。违规行为将被处以最高**3500万欧元或全球年营业额7%**的罚款（以较高者为准）。

## 禁止目录

第5条禁止以下AI系统：

1. **使用潜意识操纵**手段，实质性地影响个人行为并对其造成损害
2. **利用弱势群体的脆弱性** — 基于年龄、残疾或社会/经济状况
3. **实施社会评分** — 基于自然人的社会行为进行评估，导致不合理的不利待遇
4. **仅基于画像分析进行犯罪预测**（无具体嫌疑依据）
5. **通过无差别抓取互联网或监控摄像头数据建立人脸识别数据库**
6. **在工作场所或教育机构部署情绪识别**（例外：医疗/安全相关目的）
7. **基于敏感特征进行生物特征分类**（种族、政治观点、工会成员身份、性取向、宗教信仰）
8. **在公共场所使用实时远程生物特征识别**进行执法（严格例外：失踪人员搜寻、反恐、严重刑事犯罪）

::: info Digital Omnibus 新增的（第九项）禁止（截至2026年6月）
2026年5月7日达成临时协议的 Omnibus 改革为第5条新增了一项禁止：禁止生成或操纵**非自愿私密图像（NCII）**或**儿童性虐待材料（CSAM）**的AI系统——只要这是该系统合理可预见且可复现的结果。过渡期至**2026年12月2日**（以正式通过为准）。
:::

## BAUER GROUP 相关性

::: tip 评估结果
BAUER GROUP 目前及计划中的产品均不属于第5条禁止目录的范围。BAUER GROUP 不开发社会评分系统、工作场所情绪识别软件、生物特征分类系统、实时监控技术，也不开发生成私密或滥用性图像内容（NCII/CSAM）的系统。
:::

该评估**每年**进行一次审查，并在每次涉及AI组件的新产品开发时重新审查。
