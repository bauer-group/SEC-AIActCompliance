# 披露AI交互

## 要求

当BAUER GROUP产品实现AI系统与自然人之间的**直接交互**（例如聊天机器人、语音助手）时，必须告知该人员。

## 实施

### 聊天机器人与对话式AI

在交互开始前或开始时必须放置清晰的提示：

```
🤖 您正在与AI助手进行交流。
回复内容为自动生成，
可能包含错误。
```

### 基于API的AI服务（B2B）

对于通过API提供AI功能的B2B产品，相关提示应嵌入**技术文档和使用条款**中。

### 例外情况：显而易见性

如果从上下文中明显可以看出正在进行AI交互（例如服务名称为"BAUER AI Assistant"），则可以省略明确的提示。建议：仍然进行标注。
