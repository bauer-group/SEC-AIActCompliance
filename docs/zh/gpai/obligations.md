# GPAI提供者义务 – 第53条

## 所有GPAI模型提供者的义务

自**2025年8月2日**起，GPAI模型提供者必须：

1. **编制并维护技术文档**（附件XI）
2. **向下游提供者提供信息和文档**（附件XII），以便其理解模型并履行自身义务
3. **遵守版权指令** — 建立保护版权的政策
4. **公布训练数据摘要**（按照EU AI Office模板）

## 系统性风险的额外义务（第55条）

具有系统性风险的GPAI模型提供者还必须：

- **进行模型评估**（包括对抗性测试）
- **评估和减轻系统性风险**
- **向AI Office报告严重事件**
- **确保模型的网络安全**

## 开源例外

使用**自由开源许可证**（参数、架构和使用方式公开可访问）的GPAI模型仅需满足第3和第4项要求——**除非**其存在系统性风险。

## 对BAUER GROUP的相关性

| 问题 | 回答 |
|---|---|
| BAUER是GPAI模型提供者吗？ | **否** |
| 第53/55条适用于BAUER吗？ | **否** — 适用于模型提供者（Anthropic、OpenAI等） |
| BAUER需要核查什么？ | 所使用的GPAI提供者是否履行了其义务（尽职调查） |
| BAUER会成为提供者吗？ | 仅当以自己名义将基于GPAI的系统作为高风险系统投放欧盟市场时 |
