# 制裁制度 – Art. 99

## 罚款框架

AI Act规定了**分级制裁**，自2025年8月2日起适用：

| 违规行为 | 罚款（最高） | 替代方案 |
|---|---|---|
| **禁止的AI实践 (Art. 5)** | 3500万 EUR | 全球年营业额的7% |
| **其他义务**（高风险、透明度、GPAI等） | 1500万 EUR | 全球年营业额的3% |
| **向当局提供虚假/误导性信息** | 750万 EUR | 全球年营业额的1% |

每种情况适用**较高金额**。

### 中小企业优惠

对于中小企业和初创企业，适用上述金额中的**较低者**（绝对值 vs. 百分比）。

## 其他措施

除罚款外，市场监管机构还可以：

- 下令将AI系统**撤出**市场
- 强制执行**纠正措施**（修复、重新训练）
- **禁止**投入使用直至达到合规
- 发出**警告**

## BAUER GROUP 风险评估

罚款风险与违规行为成正比。对于BAUER GROUP：

| 场景 | 风险 | 缓解措施 |
|---|---|---|
| 禁止实践 (Art. 5) | 极高 | 每个产品的审查方案 → 风险 ≈ 0 |
| 高风险未合规 | 高 | Go/No-Go框架 → 不合规产品不投放欧盟 |
| 缺少透明度标注 | 中 | 已实施模板和流程 |
| 缺少AI素养 | 低 | 已整合到入职流程 |
