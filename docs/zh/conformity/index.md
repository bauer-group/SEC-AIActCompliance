# 合格评定 – Art. 43

## 概述

高风险AI系统必须在**投放市场前**进行合格评定。AI Act提供两条路径：

### 路径1：内部控制 (Annex VI) — 自我评估

对于**大多数**高风险AI系统，提供者的自我评估即可满足要求。包括：

- 审查QMS (Art. 17)
- 审查技术文档
- 确认符合Art. 8–15

### 路径2：第三方评估 (Annex VII)

仅对特定的**Annex III系统**要求，特别是：

- 远程生物特征识别 (Annex III Nr. 1)
- 关键基础设施 (Annex III Nr. 2) — 在特定条件下
- 某些执法系统 (Annex III Nr. 6–7)

::: tip BAUER GROUP 策略
BAUER GROUP将仅在欧盟市场投放**自我评估 (Annex VI)** 即可满足的AI系统。需要第三方合格评定的系统因不成比例的工作量而自动**No-Go**。
:::

## 受欧盟协调立法约束的产品程序

当AI系统是已受监管产品 (Annex I) 的安全组件时，AI Act合格评定将整合到**现有产品合格评定**中。
