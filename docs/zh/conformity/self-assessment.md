# 自我评估 – Annex VI

## 内部控制 (Annex VI)

自我评估是高风险AI系统的 **标准合格评定程序**（在无需第三方评估的情况下）。

### 步骤1：QMS 验证

确认 QMS（Art. 17）已实施并满足要求。

### 步骤2：技术文档

验证技术文档（Art. 11、Annex IV）完整且为最新版本。

### 步骤3：合规性检查

针对 Art. 8–15 的所有要求进行系统性检查：

| 要求 | 已检查 | 合规 | 备注 |
|---|---|---|---|
| 风险管理体系 (Art. 9) | ☐ | ☐ | |
| Data Governance (Art. 10) | ☐ | ☐ | |
| 技术文档 (Art. 11) | ☐ | ☐ | |
| 日志记录 (Art. 12) | ☐ | ☐ | |
| 透明度 (Art. 13) | ☐ | ☐ | |
| 人工监督 (Art. 14) | ☐ | ☐ | |
| 准确性、鲁棒性、网络安全 (Art. 15) | ☐ | ☐ | |

### 步骤4：欧盟合格声明

自我评估通过后：签发欧盟合格声明（Art. 47）。

### 步骤5：CE 标识

加贴 CE 标识（Art. 48）。

### 步骤6：注册

在欧盟数据库中注册（Art. 49）。
